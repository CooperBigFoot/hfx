# Specification

This directory contains the canonical development version of the HFX specification.

- [`HFX_SPEC.md`](./HFX_SPEC.md) is the normative specification under active development.
- [`CHANGELOG.md`](./CHANGELOG.md) tracks specification-level changes.

Stable published releases are expected to be cut from tagged repository versions later. Until then, this directory is the source of truth for ongoing design work.
